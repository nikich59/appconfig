# appconfig
This is a lightweight module to help you configure your application using JPA-style config objects.

<code>
allprojects { <br>
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
</code>

<code>
dependencies {
	        implementation 'com.github.nikich59:appconfig:-SNAPSHOT'
	}
</code>
